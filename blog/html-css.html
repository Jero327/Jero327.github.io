What is the difference between Margin, Border, and Padding?

Margin controls the space around an element's border, border defines the border around an element's content and padding, while padding controls the space between an element's content and its border. Together, these properties allow you to precisely control the layout and spacing of elements on a web page.


For instance:

HTML: 
            
              <h5 class="injected-text">margin</h5>

              <div class="box yellow-box">
              <h5 class="box red-box">padding</h5>
              <h5 class="box blue-box">padding</h5>
              </div>
CSS: 
            
              <style>
                .injected-text {
                  margin-bottom: -25px;
                  text-align: center;
                }

                .box {
                  border-style: solid;
                  border-color: black;
                  border-width: 5px;
                  text-align: center;
                }

                .yellow-box {
                  background-color: yellow;
                  padding: 10px;
                }

                .red-box {
                  background-color: crimson;
                  color: #fff;
                  padding: 20px;
                }

                .blue-box {
                  background-color: blue;
                  color: #fff;
                  padding: 10px;
                }
              </style>
Rendered:

There are 3 boxes: red, blue and yellow. Each background color area is filled by padding; Black color border is their border; The yellow color area between the border of red box and the border of yellow box and/or the border of blue box is defined by the margin value of red box.

When to use Margin?

You need to use Margin when controlling the transparent space outside an element's border. Margin create space between the element and other elements on the page.

How to use Margin?

There are 4 types of Margin:

margin-top
margin-right
margin-bottom
margin-left

Margin values can be set individually for each side of the element. You can define 4 directions' margin value of one box element accordingly.

When to use Border?

When you want to create border around an HTML element.

How to use Border?

CSS borders have properties like:
border-color
border-width
border-style
border-radius
border-top

                <p>

When to use Padding

Padding is the transparent space between an element's content and its border. It defines the inner spacing within an element and affects the content's position inside the element.

How to use Padding

CSS padding have properties like:
padding-top
padding-right
padding-bottom
padding-left

HTML: 

              <div class="box">
                <p>
                  This is a box
                </p>
              </div>
CSS: 
              .box {
                text-align: center;
                font-size: 40px;
                color: white;
                background-color: red;
                margin-left: 800px;
                margin-right: 800px;
                margin-top: 200px;

                padding: 20px; /*set padding 20px for all sides*/
              }
